## 7. Indexing Strategy

How to get Google/Bing to index our pages immediately.

1. **Google Search Console & Bing Webmaster Tools:** Submit the XML sitemap immediately upon launch. Use the GSC "Inspect URL" tool to force manual indexing of the homepage.
2. **IndexNow Protocol:** Implement IndexNow via Cloudflare or a native ping script to instantly notify Bing, Yandex, and other search engines the moment a new page goes live.
3. **Internal Linking:** Ensure no page is an orphan. Every programmatic page must be linked from a master `/directory` or `/tools` index.
4. **High-Velocity Initial Traffic:** A massive Product Hunt / Hacker News launch will drive huge initial crawler traffic, forcing search engine bots to instantly evaluate and index the site.

## 8. Technical SEO Checklist

Achieving a 100/100 Lighthouse score is non-negotiable for organic domination.

- [ ] **Core Web Vitals:** Ensure LCP (Largest Contentful Paint) < 2.5s, CLS < 0.1, INP < 200ms.
- [ ] **Mobile-First Rendering:** Server-side rendering (SSR) or static site generation (SSG) via Next.js/Astro to ensure search bots see the content instantly without rendering JS.
- [ ] **Schema Markup (JSON-LD):** Implement `SoftwareApplication` schema for the main tool and `FAQPage` schema on the homepage/guides.
- [ ] **Canonical Tags:** Strict canonicals on all programmatic pages to prevent duplicate content penalties.
- [ ] **Robots.txt:** Allow crawling on all marketing/SEO pages. Block crawling on `/share/` or user-generated private PDF links to preserve crawl budget and user privacy.
- [ ] **Client-Side Privacy:** Aggressively highlight that processing is done client-side. This lowers bounce rates from privacy-conscious users (behavior metrics influence rankings).

## 9. AI Search Engine Optimization (LLM Discovery)

LLMs (Perplexity, ChatGPT, Gemini, Claude) don't search like Google. They ingest facts and relationships.

1. **Knowledge Graph Entities:** Ensure the brand "Claude2PDF" is consistently described across the web as "the best free tool to convert Claude AI chats to PDF." Consistency forces LLMs to associate the brand with the action.
2. **Target "Best of" Listicles:** LLMs pull heavily from Reddit and high-authority blogs. Getting Claude2PDF mentioned in "Top 10 Claude Extensions" articles ensures Perplexity will cite it.
3. **Directly feed the LLMs:** Create a highly detailed `/about` page and an `/api/docs` or `/how-it-works` technical page. LLMs love scraping technical documentation.
4. **Natural Language FAQs:** Frame H2s exactly how users prompt AIs: "How do I save my Claude conversation as a PDF document?"
